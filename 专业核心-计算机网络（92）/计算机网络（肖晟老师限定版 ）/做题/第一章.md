# 第一章

## 做题策略

> [!tip] 使用方式
> 第一章不需要大面积刷题。这里按《计算机网络：自顶向下方法》第八版书后题校准：R 题用于补简答/填空，P 题只做能训练期末应用题的计算题。视频只在计算题看答案还不懂时再开。

1. 先做 R 题，目标是能说出关键词。
2. 再做 P 题，目标是会算时延、吞吐量、丢包概率和分组交换过程。
3. 做法：先自己做，再看答案；答案看不懂时再看中科大视频对应讲解。

## 第八版 R 题怎么做

### 必做口答

- [ ] **R1**：主机/端系统、端系统例子，Web 服务器是否是端系统。
- [ ] **R3**：标准对协议的重要性。
- [ ] **R4、R5、R8、R9、R10**：接入网和物理媒介，重点 DSL/HFC/FTTH、以太网、无线接入。
- [ ] **R11**：一台存储转发交换机、两段链路 $R_1/R_2$，端到端时延。
- [ ] **R12、R13**：电路交换 vs 分组交换、TDM/FDM、统计多路复用概率。
- [ ] **R14、R15**：ISP 对等、IXP、内容提供商自建网络。
- [ ] **R16、R18、R19**：四类时延、传输/传播时延、吞吐量瓶颈。
- [ ] **R20**：长文件如何分组，交换机用目的地址/转发表决定输出链路。
- [ ] **R22-R25**：协议分层、五层协议栈、PDU 名称、路由器/交换机/主机处理层次。
- [ ] **R26-R28**：病毒/蠕虫、僵尸网络/DDoS、中间人攻击能做什么。

### 可跳或只看答案

- **R2**：外交协议类比，太偏。
- **R6**：本地接入价格调研，期末不考。
- **R7、R17、R21**：官网交互动画题，不适合作为纸笔复习。

### R 题做法

- 每题先自己口答 30 秒。
- 写不出就直接看答案，不要耗太久。
- 只把容易忘的关键词补到 A4。

## 第八版 P 题推荐

### 必做

- [ ] **P2**：多个分组连续经过 $N$ 段链路，练流水线公式：$(N+P-1)L/R$。
- [ ] **P3**：稳定速率应用适合电路交换还是分组交换；若总速率小于链路容量，是否需要拥塞控制。
- [ ] **P6**：传输时延 $L/R$ 和传播时延 $m/s$ 的核心题，必须会。
- [ ] **P8 或 P9**：分组交换 vs 电路交换用户容量、二项分布概率。二选一认真做即可。
- [ ] **P10**：三段链路端到端时延，练逐跳相加。
- [ ] **P12**：排队时延具体计算：正在传半个分组 + 队列已有多个分组。
- [ ] **P14、P15**：流量强度 $I=La/R$ 与排队+传输总时延，P15 是把公式改写成 $1/(\mu-a)$。
- [ ] **P20**：多对客户-服务器共享瓶颈链路，练吞吐量 $\min\{R_s,R_c,R/M\}$。
- [ ] **P25 或 P27**：带宽-时延积、管道中的比特数、一个比特的空间宽度。二选一。
- [ ] **P31**：报文分段为什么降低时延，顺便记分段的优缺点。

### 可选

- [ ] **P5**：车队类比，适合还没分清传播/传输时延时做。
- [ ] **P7**：VoIP 分组化时延，算“收集分组时间 + 传输 + 传播”。
- [ ] **P13**：一批分组同时到达时的平均排队时延。
- [ ] **P16**：Little 公式，偏排队论，有余力再做。
- [ ] **P19**：Metcalfe 定律，偏概念拓展。
- [ ] **P21、P22、P23**：吞吐量/丢包/packet pair，更偏拓展，期末优先级低于 P20。
- [ ] **P24、P26、P28、P29**：大文件传输、FedEx 类比、卫星链路，能补带宽-时延积但不必全做。

### 不建议正式做

- **P1**：ATM 应用层协议设计，偏开放题。
- **P4**：图 1-13 电路交换网络连接数，若老师没强调图可跳。
- **P18**：Traceroute 实验题，不适合纸笔期末。
- **P30**：航空旅行类比分层首部，偏开放。
- **P32**：网站动画实验题，可跳。

### P 题做法

- 每题先自己做 5-8 分钟。
- 卡住时先看答案，不要立刻开视频。
- 答案仍看不懂，再看中科大视频对应讲解。
- 第一章视频主要给时延、吞吐量、分组交换推导服务，概念题不需要看视频。

## 视频使用规则

- **R 题**：默认不看视频，自己口答后对答案即可。
- **P 题**：先自己做，再看答案；如果答案仍看不懂，再看中科大对应讲解。
- 第一章视频主要给时延、吞吐量、分组交换推导服务，概念题不需要看视频。
